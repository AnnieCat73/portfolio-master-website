GitHub Repo Gallery

A Vanilla Javascript project displaying a gallery of my repos from my GitHub portfolio

VISIT THE WEBSITE

Did you know

  • 1. In a nutshell, GitHub is a web-based version-control and collaboration platform for developers.
  • 2. The GitHub Repo Gallery is a repository or repo where you can access all its files and versions that have been saved.
  • 3. I use GitHub every day to track changes in the projects I code.

The task

GitHub Repo Gallery is a project that pulls data from my GitHub portfolio to create a gallery of all my repos.

Any visitors to the GitHub Repo Gallery site can click on any repos to see more details of each of my repos.

The main focus of the project is Vanilla Javascript.The HTML and CSS basic structure are by Skillcrush, one of my course providers.

How I did this

To make the GitHub Repo Gallery functional, this is what I did:

  • 1. I ‘spruced up’ my GitHub profile and repos and added a couple of projects.
  • 2. I forked and cloned the files for the project.
  • 3. I studied the set-up to work out how I would approach it with Javascript.
  • 4. Once I had my plan in place I fetched the data from the GitHub API.
  • 5. I uploaded the GitHub Repo Gallery onto the web via GitHub Pages.

What I learnt

  • 1. Git and GitHub – This was an opportunity to use and get more comfortable with the many the features of Git and GitHub.
  • 2. API JSON Data – I got to practice with fetching information from an API and integrate it into the project.
  • 3. Javascript – I used Javascript features such as global variables, click events, functions, arrays, the matches() method and DOM manipulation to complete the functionality of the project.
